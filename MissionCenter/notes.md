# 筆記

## 開放問題

- 第一版預設輸出 FPS 暫定 12 或 15，實作前可依範例素材調整。
- 最大單張 Sprite Sheet 寬度暫定 4096 或 8192，需以 Godot 匯入體驗驗證。
- 髮絲模式候選：rembg、transparent-background、RVM、MatAnyone 2；MVP 優先選依賴與授權最容易落地者。

## 觀察

- 使用者主要想處理網頁生成影片，目標素材包含人物動作與建築變形。
- 複雜背景不是第一版目標；背景應盡量是白、黑、綠或接近純色。
- 匯出後需要直接在頁面確認成果，避免每次都打開 zip 才知道 sheet 是否能用。
- 右下角星星若是 Google Veo 浮水印，工具會保留來源內容，不做自動移除。
- 公開 README 需維持一般使用者視角，不放入對話中的私人稱呼或角色設定。
